EventList 1.0.2 Stable Help

Welcome

Welcome to the stable version of Eventlist 1.0.2 designed for the Joomla! 1.6 or better. Basically, this version of EventList is a port of the last stable version ( EventList 1.0.1 for Joomla 1.5.X) into the newest version of Joomla! (version 1.6.0 stable or better)

The goal of this port was to get this extension into 1.6 + with minimal code changes. There are really no new features from an end-user perspective; the changes are mainly "under-the-hood" to allow Eventlist to function. As well, this version does not take advantage of the rich set of improvements in Joomla! - most notably ACL.

As such, you should expect that the known issues in V 1.0.1 most likely will be encountered in this version. There may be new issues related to interaction with Joomla!

The EventList and EventList Wide module have been updated, as well as the Mailer and Searchbot plugins. They have the same version numbers as this stable release (1.0.2), but are distributed as separate installs.

All other addons, modules, plugins and extensions for EventList may not yet been ported to Joomla! 1.6+ so they most likely will not work. This release of EventList will allow others to develop and upgrade those solutions into the new Joomla! version.

The main areas that were changed are:

  • Modifications needed to get the pages (views) to dispay in J1.6+,
  • Updating of all the 1.0.1 language files (28 in all) to meet Joomla!'s new translation format,
  • Modifications to get the permissions for the Public, Registered and Special groups working in J1.6+,
  • Use of the new language features to support German translations during the install,
  • New sample data images, and
  • Minor changes in the help to reflect the above.

    • We hope you enjoy this version of EventList and Joomla! 1.6+!.

    Introduction

    EventList is just that what the name would lead you to expect. It's not a calendar but an easy way to manage and present your Events for your group, company or community. EventList offers a lot features and possibilities which makes your work easier, like the Venue database, Frontend editing with more than the three Joomla! access levels, Picture resizing and much more. But also for your site visitors EventList has some additional features to offer, like a simple registration system, map service links, or the submission of new events and venues.

    Quick Start guide


    After the successful installation of EventList you need to do at least 3 steps for the system to work.

    1. The first one is to configure the settings you need in the EventList configuration. Set the format of the date and time, enable image thumbnail creation, set the name of the table columns or integrate the community builder enhanced user profile component. Most of the settings are self explanatory. If not, move your mouse pointer over the text to get more detailed information or click on the help icon in the toolbar.


    2. The second step is to create the categories. Every event you create must be assigned to a category. For example, if you have a music site you could use Rock, Ska, Electronic, etc. For each category you can set access permission levels. All events assigned to this category will only be visible for the group you have given access to.


    3. The third step is to create the menu item in the Joomla! menu manager. EventList offers 11 different menu views:
    • A simple List of events
    • A category listing
    • A category listing with limited number of events as preview
    • A listing of the current day
    • A detailed Venue listing
    • To a specified Event
    • To a category with all events assigned to it
    • To a Venue with all events assigned to it
    • To the event submission form
    • To the venue submission form
    • A simple List events and registrations the user submitted


    If you have done these steps you are ready to type in your events. If you want to assign your events to venues you have first to create the venue information. If you like EventList and find it useful, please consider a donation.
